### Instal

```sh
npm install @casual-ui/react
```

### Usage

- Import styles in client entry file

```js
import '@casual-ui/react/dist/style.css'
```

- Use in jsx

```jsx
import { CButton } from '@casual-ui/react'

const APP = () => <CButton label="A Button" />
```
